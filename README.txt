Wordpress plugin that register widget to display random different car photo and description.




== Description ==

This plugin register custom post type like 'cars'.
Register widget to display (random) these cars on site.
After your first activation of plugin, Demo Content will be insert only once.
Then After second activation\deactivation of plugin Demo Content will not be insert,
but you can manually create new car custom post in admin or edit exist demo cars;
